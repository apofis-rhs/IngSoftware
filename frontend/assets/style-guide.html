Guía de Estilos — LUMIKA

Referencia visual del equipo. Antes de escribir CSS nuevo, revisa si ya existe el componente aquí.

📏 Reglas obligatorias

✅ Importa siempre los 3 CSS en este orden: variables.css → base.css → components.css
❌ Nunca escribas colores, fuentes o tamaños directos. Siempre var(--nombre)
❌ Nunca escribas fetch() en tu pantalla. Importa de api.js
✅ Copia TEMPLATE-PANTALLA.html para empezar cada pantalla nueva
✅ Tu CSS solo tiene estilos únicos de tu pantalla. Lo demás ya está en los 3 base
